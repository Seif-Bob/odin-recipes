Description
This project is part of the The Odin Project's Foundations Course. It is a basic recipe website built using HTML only. The primary goal of this project is to practice and demonstrate fundamental web development skills before moving on to CSS and more complex layouts.

Skills Demonstrated
By completing this project, I have practiced the following:

HTML Structure: Setting up a standard boilerplate.

Text Formatting: Using headings, paragraphs, and emphasis tags.

Lists: Organizing content using both ordered and unordered lists.

Links & Navigation: Implementing internal links to navigate between multiple HTML pages.

Images: Adding and properly attributes images to enhance web content.

Git & GitHub Workflow: Managing the project through local commits and pushing to a remote repository.# odin-recipes
